# BareBnB Plus

Travel anywhere and the world and experience a hassle-free booking entourage. Stay at a room next to the ocean in Waikiki for a fraction the cost of a hotel.

This module allows the user to book at world-class homes, apartments, and rooms across the world in an easy fashion.
